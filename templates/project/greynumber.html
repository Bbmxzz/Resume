Code Converter

Receive input from all 4 switches as a 4-bit binary number, then display the input as a hexadecimal value on a 7-segment display. If the value is a prime number, an LED will light up. Additionally, the value is displayed in Grey code through LEDs, and both the binary and Grey code values are shown on NETPIE.


Responsible for wiring the circuit, programming and connecting it to display on NETPIE.
