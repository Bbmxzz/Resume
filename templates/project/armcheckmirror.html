Arm Check Mirror

The ADXL345 sensor is connected to an ESP8266 Mini. When the user raises their arm, the sensor detects values on all three axes and sends this data to the ESP8266 Mini, which converts it into degrees. The information is then saved in real-time on Firebase. After that, the values can be pulled from Firebase and shown on a computer screen using Raspberry Pi. This project can be used for medical purposes.
