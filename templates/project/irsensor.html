Eye-to-Computer Screen Distance Detector

The circuit is set up as an IR sensor circuit to detect nearby objects. It detects when the user is sitting in front of the computer and starts counting the time spent there. If the time exceeds a set limit, it triggers an alert. This setup is also connected to NETPIE, allowing the results to be displayed on the computer screen.


Responsible for wiring the circuit , programming and connecting it to NETPIE.
